# v1.4 — Agent 智能助手

:::caution 测试阶段
v1.4 目前仍处于测试阶段，以下功能和 API 可能会在正式发布前进行调整。欢迎试用并反馈问题。
:::

v1.4 是脚本猫的一次重大更新，引入了全新的 **Agent 智能助手**系统，让用户可以通过自然语言对话来自动化浏览器操作、提取网页数据、管理文件和执行定时任务。

## Agent 智能助手

在管理页面新增了 Agent 聊天入口，可以与 AI 模型进行多轮对话，AI 会自动调用内置工具完成任务。

**核心能力：**

- **多模型支持** — OpenAI 兼容格式和 Anthropic 原生格式，支持 Prompt Caching
- **11 个内置工具** — 网页抓取/搜索、标签页管理、OPFS 文件系统、代码执行、用户交互等
- **Skill 系统** — 提示词 + 工具脚本 + 参考资料的能力包，可扩展 Agent 专业领域知识
- **MCP 集成** — 支持 [Model Context Protocol](https://modelcontextprotocol.io/)，连接外部工具服务器
- **DOM 自动化** — 页面导航、截图、点击、填充、滚动、脚本执行等
- **定时任务** — 基于 Cron 表达式的定时任务调度（Internal / Event 模式）
- **子代理** — 可生成独立的子代理处理复杂子任务
- **上下文管理** — Auto-compact、Prompt Caching，减少 Token 消耗

**脚本 API（`CAT.agent.*`）：**

用户脚本可通过 `CAT.agent.*` 命名空间调用 Agent 能力，包括 conversation、dom、skills、task、model、opfs、mcp 等 API。每个 API 需要通过 `@grant` 声明对应权限。

**6 个管理页面：** 聊天、模型配置、Skill 管理、定时任务、MCP 服务器、OPFS 浏览器。

## 其他新功能

- **支持 `@unwrap` 标签** — 允许脚本直接注入页面
- **`window.onurlchange`** — 以 Navigation API 实现 Tampermonkey 兼容的 URL 变化事件
- **编辑器改进** — 新增编辑菜单（查找、替换、撤销等），修复多个快捷键和提示问题
- **多平台搜索引擎** — 获取脚本支持选择不同搜索引擎
- **更多图标供应商** — 新增更多脚本图标服务供应商
