# v1.3

## New Script Installation Flow [#842](https://github.com/scriptscat/scriptcat/issues/842)

Redesigned the script installation flow — no longer relies on external website redirects or opening new windows. Scripts are now installed directly within the current page. The install page layout has also been optimized with responsive design, scrollable code preview, and inline error display for download failures.

## Script Run Timing Options [#895](https://github.com/scriptscat/scriptcat/issues/895)

Added a run timing option in script settings, allowing you to choose from `document-start`, `document-body`, `document-end`, `document-idle`, `early-start` and more, giving you precise control over when your script executes.

![Script run timing options](v1.3.assets/run-timing.png)

## Amazon S3 Storage Support [#1146](https://github.com/scriptscat/scriptcat/issues/1146)

Cloud sync and backup now supports Amazon S3 as a storage backend, in addition to existing options like WebDAV, Baidu Netdisk, OneDrive, Google Drive, and Dropbox.

![Amazon S3 storage option](v1.3.assets/s3-storage.png)

## Cloud Storage Unbinding [#1151](https://github.com/scriptscat/scriptcat/issues/1151)

Added an unbind button in cloud sync settings, making it easy to disconnect or switch cloud storage connections.
