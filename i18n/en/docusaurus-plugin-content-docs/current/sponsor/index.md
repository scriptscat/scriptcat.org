ScriptCat is an open-source project maintained by volunteers, with no company backing it. Servers, domains and other infrastructure, along with a great deal of development and maintenance time, are all sustained by community support. Every contribution helps ScriptCat go further.

## Where your sponsorship goes

Sponsorship income mainly goes toward:

- **Covering running costs** — servers, domains and other day-to-day infrastructure.
- **Rewarding maintainers & contributors** — so long-term maintainers, and everyone who sends code, translations and docs, get the reward they deserve.
- **Building new features** — more time to polish the experience and develop new capabilities.

## How to sponsor

Whether you're an individual or a company, and no matter the size, every bit of support counts.

We collect sponsorships through [OpenCollective](https://opencollective.com/scriptcat), with one-time or monthly contributions; your name will appear on the backers list after you donate.

If you'd like a **sponsor placement** — your logo and intro shown in the README, on the website or in the docs — email us at [codfrm@gmail.com](mailto:codfrm@gmail.com).

## Sponsor placements

A sponsor's logo and description can be shown in the following places, arranged via email:

| Placement | Format |
| --- | --- |
| Top of the GitHub README | Large banner |
| GitHub README sponsors area | Logo + one-line intro |
| Website sponsor page (this page) | Logo + intro |
| Website docs pages (e.g. changelog) | Logo display |
