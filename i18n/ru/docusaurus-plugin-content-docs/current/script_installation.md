---
title: Установка скрипта
---

Эта страница используется как промежуточный шаг при установке нового пользовательского скрипта в `ScriptCat`.

После завершения установки страница должна автоматически закрыться или выполнить перенаправление.

Страница не предназначена для прямого просмотра. При необходимости откройте [Установка скриптов](/docs/use/script_installation/).
