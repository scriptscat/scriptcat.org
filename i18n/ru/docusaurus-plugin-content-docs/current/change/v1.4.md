v1.4 — крупное обновление ScriptCat с новой системой **Agent**: автоматизация действий в браузере, извлечение данных со страниц, работа с файлами и отложенные задачи через диалог на естественном языке.

## Agent

В панели управления появился чат с Agent: многоходовой диалог с ИИ-моделью; ИИ сам вызывает встроенные инструменты.

**Основные возможности:**

- **Несколько моделей** — форматы, совместимые с OpenAI, и нативный Anthropic, с поддержкой Prompt Caching
- **11 встроенных инструментов** — fetch/поиск в вебе, вкладки, файловая система OPFS, выполнение кода, взаимодействие с пользователем и др.
- **Система Skill** — пакеты промптов + скрипты инструментов + справочные материалы, расширяющие знания Agent в предметной области
- **Интеграция MCP** — поддержка [Model Context Protocol](https://modelcontextprotocol.io/) для внешних серверов инструментов
- **DOM-автоматизация** — навигация, скриншоты, клики, заполнение, прокрутка, выполнение скриптов и др.
- **Отложенные задачи** — планирование по Cron (режимы Internal / Event)
- **Субагенты** — независимые субагенты для сложных подзадач
- **Управление контекстом** — Auto-compact и Prompt Caching для снижения расхода токенов

**Script API (`CAT.agent.*`):**

Пользовательские скрипты вызывают возможности Agent через пространство имён `CAT.agent.*`: conversation, dom, skills, task, model, opfs, mcp. Для каждого API нужно объявить соответствующее разрешение через `@grant`.

**6 страниц панели:** Chat, Model Configuration, Skill Management, Scheduled Tasks, MCP Servers и OPFS Browser.

## Поддержка тега `@unwrap`

Позволяет внедрять скрипт прямо в страницу, минуя sandbox-обёртку.

## Событие `window.onurlchange`

Совместимое с Tampermonkey отслеживание смены URL на базе Navigation API.

## Улучшения редактора

Добавлено меню правки (поиск, замена, отмена и т.д.), исправлены несколько проблем с горячими клавишами и подсказками.

## Поисковые движки разных платформ

Поиск скриптов поддерживает выбор разных поисковых платформ.

## Больше провайдеров иконок
