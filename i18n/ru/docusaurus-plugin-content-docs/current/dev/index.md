Это расширение поддерживает три типа скриптов:

- Фоновые скрипты страницы (foreground)
- [Фоновые скрипты](./background.md)
- [Скрипты по расписанию](./background.md#scheduled-script-crontab)

Фоновые скрипты страницы — то, что обычно имеют в виду под скриптами Tampermonkey. Фоновые скрипты и скрипты по расписанию — типы, специфичные для расширения ScriptCat; подробности смотрите в соответствующей документации.

С помощью этой документации вы можете изучить [API Tampermonkey](./api.md) и [фоновые скрипты ScriptCat](./background.md). Мы также поддерживаем сайт [руководства по разработке скриптов Tampermonkey](https://learn.scriptcat.org/) с большим количеством учебных материалов — добро пожаловать к обсуждению там.

Если вы заметите пробелы в документации, сообщите нам на [GitHub](https://github.com/scriptscat/scriptcat.org) или отправьте PR, чтобы помочь её улучшить.
