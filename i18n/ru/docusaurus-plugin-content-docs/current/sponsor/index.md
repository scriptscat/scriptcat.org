ScriptCat — открытый проект, поддерживаемый волонтёрами, без корпоративной поддержки. Серверы, домены и прочая инфраструктура, а также большое количество времени на разработку и сопровождение обеспечиваются поддержкой сообщества. Каждый вклад помогает ScriptCat развиваться дальше.

## Куда идут средства спонсорства

Доходы от спонсорства в основном идут на:

- **покрытие операционных расходов** — серверы, домены и повседневная инфраструктура;
- **вознаграждение мейнтейнеров и участников** — чтобы долгосрочные мейнтейнеры и все, кто вносит код, переводы и документацию, получали заслуженную поддержку;
- **разработку новых возможностей** — больше времени на полировку пользовательского опыта и создание новых функций.

## Как стать спонсором

Неважно, частное вы лицо или компания, и каков размер поддержки — каждый вклад имеет значение.

Мы принимаем спонсорские взносы через [OpenCollective](https://opencollective.com/scriptcat): разовые или ежемесячные; после пожертвования ваше имя появится в списке спонсоров (backers).

Если вам нужен **спонсорский плейсмент** — логотип и описание в README, на сайте или в документации — напишите нам на [codfrm@gmail.com](mailto:codfrm@gmail.com).

## Спонсорские плейсменты

Логотип и описание спонсора могут отображаться в следующих местах (по договорённости по электронной почте):

| Место | Формат |
| --- | --- |
| Верх GitHub README | Большой баннер |
| Блок спонсоров в GitHub README | Логотип + однострочное описание |
| Страница спонсоров на сайте (эта страница) | Логотип + описание |
| Страницы документации на сайте (например, changelog) | Отображение логотипа |
